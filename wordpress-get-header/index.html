wordpress - get_header()

描述

从当前主题中引入 header.php 模板文件。如果名字是特定的，那么包含特定名称的头部文件 header-{name}.php 就会被引入。 如果主题没有 header.php 文件，就会引入默认文件 wp-includes/theme-compat/header.php 。

用法

参数

$name

(string) (_可选_) 调用 header-name.php.

默认: None
